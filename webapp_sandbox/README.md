# 🎨 LegalTax Design Sandbox

Тут ви можете безпечно редагувати дизайн, експериментувати та створювати нові сторінки!

## Як це працює:
1. Ви робите будь-які зміни в HTML/JS/CSS файлах **у цій папці (`webapp_sandbox`)**.
2. Коли ви захочете перенести свої зміни у справжній робочий додаток, ви можете просто написати мені:
   * *"Перенеси зміни з сендбоксу"*
   * Або запустити скрипт `apply_sandbox.py` в корені проекту.

Успіхів у дизайні! 🚀
